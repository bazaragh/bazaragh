# Bazar AGH

## Prerequisites

- Python 3.10

## Getting started

### Create virtual environment

```bash
python3 -m venv ./venv
source venv/bin/activate
```

### Install requirements

```bash
pip install -r requirements.txt
```

### Start application

```bash
flask run
# or
FLASK_ENV=development FLASK_DEBUG=1 flask run
```

Your application is running on `http://127.0.0.1:5000`
